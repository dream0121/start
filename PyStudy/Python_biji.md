# 2.1Python语法规则
## 2.1.1 注释
### 2.1.1.1 单行注释
```
# 这是一个单行注释
```
### 2.1.1.2 多行注释
```
'''
这是多行注释
可以包含多行文本
'''
```
### 2.1.1.3 中文编码声明注释
```
# coding=编码格式
```
## 2.1.2 代码缩进
> 采用代码缩进和冒号区分代码之间的层次
> > 空格键或Tab键实现，一般用 4 个空格，Tab 键为 4 个空格
## 2.1.3 编码规范
### 2.1.3.1 编写规则
> Python中采用 PEP 8 编码规范，Python Enhancement Proposals，即 Python 增强建议书.
> 1. 每个import语句只导入一个模块
> 2. 不要在行尾添加分号“；“
> 建议每行不超过 80 字符，如果超过，建议使用小括号将多行内容隐式地连接起来。
### 2.1.3.2 命名规范
> 1. 模块名尽量短小，全部小写，可以使用下划线来提高可读性；如：game_main
> 2. 包名尽量短小，全部小写，不建议使用下划线,如：com.example.game
> 3. 类名采用单词首字母的大写形式,如：BorrowBook
> 4. 模块内部的类采用下划线加类名，如：_BorrowBook
> 5. 函数、类的属性和方法的命名规则同模块类似，也是全部使用小写字母，多个字母之间用下划线连接，如：get_book_name
> 6. 常量命名时采用全部大写，可以使用下划线
> 7. 使用双下划线开头的实例变量或方法是类私有的。


# 2.2 保留字与标识符
## 2.2.1 保留字
> Python中的保留字是指在Python语言中已经被赋予特殊含义的单词，不能作为变量名、函数名或其他标识符的名称。

|and|as|assert|break|class|continue|
|----|----|----|----|----|----|
|def|del|elif|else|except|execfinally|
|for|from|False|global|if|import|
|in|is|lambda|nonlocal|not|none|
|or|pass|raise|return|try|True|
|while|with|yield|

> 注意：Python中的保留字是区分大小写的。
> Python中的保留字可以通过在python shell中输入代码来查看。
> ```python
> import keyword
> keyword.kwlist
> ```
## 2.2.2 标识符
> 标识符是指在Python语言中用于标识变量、函数、类、模块等的名称。
> 标识符的命名规则如下：
> 1. 标识符只能包含字母、数字和下划线“_”，不能以数字开头。
> 2. 标识符不能使用Python中的保留字作为名称。
> 3. 标识符的命名应该具有可读性，尽量使用有意义的名称。
> 4. 标识符的命名应该遵循大小写敏感的原则，即区分大小写。
> 5. python中以双下划线开头的标识符有特殊意义，一般应避免使用相似的标识符。
>

# 2.4 基本数据类型
## 2.4.1 数字类型
### 2.4.1.1 整数
>正整数、负整数、0
> 十进制、八进制、十六进制和二进制
>
> 八进制以0o或0O开头，如：0o10
> 十六进制以0x或0X开头，如：0x10

### 2.4.1.2 浮点数
> 浮点数是小数，可以用科学计数法表示，如：3.14e-2

### 2.4.1.3 复数
> 复数是由实数部分和虚数部分组成的数，可以用a+bi表示，其中a和b是实数部分，i是虚数单位。
>

## 2.4.2 字符串类型
> 字符串是用单引号或双引号括起来的字符序列。
> 字符串可以包含字母、数字、空格、标点符号等。
> 字符串可以使用三引号来表示多行字符串，如：'''多行字符串'''。
> 转义字符：使用反斜杠对一些特殊字符进行lating，如：\n表示换行符。
> 
> ***在字符串定界符引号的前面加上字母r，可以表示原始字符串，原始字符串中的转义字符不会被解释，如：r'原始字符串'。***


## 2.4.3 布尔类型
> 布尔类型只有两个值：True和False。
> 布尔类型可以用于条件判断，如：if语句。
> 布尔类型可以用于逻辑运算，如：and、or、not等。


## 2.4.4 数据类型转换
> 虽然在python中不需要显式地声明变量类型，但可以使用内置函数type()来查看变量的类型。
> 
> ***int(X)***：将X转换为整数类型
> ***float(X)***：将X转换为浮点数类型
> ***complex(real[,imag])***：创建一个复数，real是实数部分，imag是虚数部分，默认imag为0。
> ***str(X)***：将X转换为字符串类型

# 2.5 输入与输出
## 2.5.1 input()函数
> input()函数用于从控制台获取用户输入，并将其转换为字符串类型。
```python
name = input("请输入你的名字：")
age = int(input("请输入你的年龄："))
```

## 2.5.2 print()函数
> print()函数用于在控制台输出信息。
```python
print("你好，", name, "！")
print("你今年", age, "岁。")
```
> ***print()函数可以使用分号分隔多个输出项，也可以使用逗号分隔多个输出项，逗号分隔的输出项之间会自动添加一个空格。***
> 每一条print()语句会在输出末尾自动添加一个换行符，如果不想换行，可以使用end参数来指定输出的结束符，如：print("Hello", end="")。

> **print()输出到文件**
> ```python
> fp =  open("output.txt", "w")
> print("Hello, world!", file=fp)
> fp.close()
> ```

# 3 运算符与表达式
## 3.1 算术运算符
> 加法运算符：+
> 减法运算符：-
> 乘法运算符：* 
> 除法运算符：/
> 模运算符：% **求余**,**如果除数是负数，结果也是负数**
> 乘方运算符：** （**表示乘方）X 的 Y 次方
> 取整运算符：// **商的整数部分**

> 算术运算符的优先级
> 第一级：**乘方运算符**
> 第二级：**乘法运算符** 、 **除法运算符** 、 **求余** 、 **取整运算符**
> 第三级：**加法运算符** 、 **减法运算符**

## 3.2 赋值运算符
> 赋值运算符用于给变量赋值，如：=
> += ：将变量加上右侧值
> -= ：将变量减去右侧值
> *= ：将变量乘以右侧值
> /= ：将变量除以右侧值
> %= ：将变量求余，并赋值给变量
> **= ：将变量乘方，并赋值给变量
> //= ：将变量除以右侧值，并取整，并赋值给变量

### 3.2.1 多重赋值、交换赋值
> 多重赋值：同时给多个变量赋值，如：a, b = 1, 2
> 

> 交换赋值：交换两个变量的值，如：a, b = b, a 或 a = b = c = 1000
> ```python
> a = 1
> b = 2
> a, b = b, a
> print(a, b)  # 输出结果：2 1
> ```

## 3.3 比较运算符(关系运算符)

> 比较运算符用于比较两个值的大小，返回布尔值。
> == ：等于
> != ：不等于   
>  \> ：大于
>  \< ：小于
>  \>= ：大于等于
>  \<= ：小于等于

## 3.4 逻辑运算符
> 逻辑运算符用于对布尔值进行逻辑操作，返回布尔值。
> and ：与
> or ：或
> not ：非

## 3.5 位运算符
> 位运算符用于对二进制位进行操作，返回整数。
> & ：按位与 **两个位都为1时，结果才为1**
> | ：按位或 **两个位都是0时，结果才为0**
> ^ ：按位异或 **两个位不同时，结果才为1**
> ~ ：按位取反 **反转所有位**
> << ：左移 **将数字的二进制位全部左移若干位，右边用0填充，左边丢弃**
> \>> ：右移 **将数字的二进制全部右移若干位，左边用0填充，右边丢弃**

## 3.6 赋值表达式
> 使用“：=“运算符实现，用于在表达式内部为变量赋值，它被称为“海象运算符“。
> 主要用于降低程序的复杂性，并提升可读性。
> ```python
name = "小明"
if (n:=len(name))>1:
    print("名字长度大于1")
```
