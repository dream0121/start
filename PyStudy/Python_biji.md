## 2.1.3 编码规范
### 2.1.3.1 编写规则
> Python中采用 PEP 8 编码规范，Python Enhancement Proposals，即 Python 增强建议书.
> 1. 每个import语句只导入一个模块
> 2. 不要在行尾添加分号“；“
> 建议每行不超过 80 字符，如果超过，建议使用小括号将多行内容隐式地连接起来。
### 2.1.3.2 命名规范
> 1. 模块名尽量短小，全部小写，可以使用下划线来提高可读性；如：game_main
> 2. 包名尽量短小，全部小写，不建议使用下划线,如：com.example.game
> 3. 类名采用单词首字母的大写形式,如：BorrowBook
> 4. 模块内部的类采用下划线加类名，如：_BorrowBook
> 5. 函数、类的属性和方法的命名规则同模块类似，也是全部使用小写字母，多个字母之间用下划线连接，如：get_book_name
> 6. 常量命名时采用全部大写，可以使用下划线
> 7. 使用双下划线开头的实例变量或方法是类私有的。


# 2.2 保留字与标识符
## 2.2.1 保留字
> Python中的保留字是指在Python语言中已经被赋予特殊含义的单词，不能作为变量名、函数名或其他标识符的名称。

|and|as|assert|break|class|continue|
|----|----|----|----|----|----|
|def|del|elif|else|except|execfinally|
|for|from|False|global|if|import|
|in|is|lambda|nonlocal|not|none|
|or|pass|raise|return|try|True|
|while|with|yield|

> 注意：Python中的保留字是区分大小写的。
> Python中的保留字可以通过在python shell中输入代码来查看。
> ```python
> import keyword
> keyword.kwlist
> ```
## 2.2.2 标识符
> 标识符是指在Python语言中用于标识变量、函数、类、模块等的名称。
> 标识符的命名规则如下：
> 1. 标识符只能包含字母、数字和下划线“_”，不能以数字开头。
> 2. 标识符不能使用Python中的保留字作为名称。
> 3. 标识符的命名应该具有可读性，尽量使用有意义的名称。
> 4. 标识符的命名应该遵循大小写敏感的原则，即区分大小写。
> 5. python中以双下划线开头的标识符有特殊意义，一般应避免使用相似的标识符。

# 5 序列的通用操作
## 5.1 索引(indexing)
**序列是一块用于存放多个值的连续内存空间，并且按一定的顺序排列，每个值都可以通过一个整数索引来访问。**
> 在 Python 中，序列的索引是从0开始的，可以使用正索引和负索引。
> 正索引：从左到右，从0开始。
> 负索引：从右到左，从-1开始。

## 5.2 切片(slicing)
**切片是一种从序列中提取子序列的方法，语法如下：**
> ```python
> sequence[start:end:step]
> ```
> start：开始索引（包含）
> end：结束索引（不包含）
> step：步长（默认为1）

## 5.3 序列加法(Adding)
**序列加法是指将两个序列连接在一起，语法如下：**
> ```python
> sequence1 + sequence2
> ```
> 返回一个新的序列，包含sequence1和sequence2的所有元素。

## 5.4 序列乘法(Multiplying)
**序列乘法是指将一个序列重复多次，语法如下：**
> ```python
> sequence * n
> ```
> 返回一个新的序列，包含sequence的n个副本。

## 5.5 检查某个元素是否是序列的成员（元素）
**检查某个元素是否是序列的成员，语法如下：**
> ```python
> element in sequence
> ```
> 返回一个布尔值，如果element是sequence的成员，则返回True，否则返回False。

## 5.6 序列的长度、最大值、最小值
**获取序列的长度、最大值和最小值，语法如下：**
> ```python
> len(sequence)
> max(interable,default=obj, key=func)
> min(interable,default=obj, key=func)
> ```

## 6.6 列表的排序
对于列表的排序，可以使用sort()方法（**列表的方法**）和sorted()函数（**系统自带的函数**）。
> sort()方法：对列表进行原地排序，语法如下：list1.sort(key=func, reverse=flag)
> reverse=flag：如果为True，则降序排序，否则升序排序。默认为False。
> key=func：指定排序的依据，func是一个函数，返回一个可比较的值。默认为None。  **如：“key=str.lower”表示将列表中的元素转换为小写进行排序**
> sorted()函数：返回一个新的排序后的列表，语法如下：sorted(list1)

> sorted()函数和sort()方法的区别：
> 1. sorted()函数返回一个新的排序后的列表，而sort()方法对列表进行原地排序。
> 2. sorted()函数可以指定排序的依据，而sort()方法只能使用key参数。
> 3. sorted()函数可以指定是否降序排序，而sort()方法只能使用reverse参数。
