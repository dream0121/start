# 2.1Python语法规则
## 2.1.1 注释
### 2.1.1.1 单行注释
```
# 这是一个单行注释
```
### 2.1.1.2 多行注释
```
'''
这是多行注释
可以包含多行文本
'''
```
### 2.1.1.3 中文编码声明注释
```
# coding=编码格式
```
## 2.1.2 代码缩进
> 采用代码缩进和冒号区分代码之间的层次
> > 空格键或Tab键实现，一般用 4 个空格，Tab 键为 4 个空格
## 2.1.3 编码规范
### 2.1.3.1 编写规则
> Python中采用 PEP 8 编码规范，Python Enhancement Proposals，即 Python 增强建议书.
> 1. 每个import语句只导入一个模块
> 2. 不要在行尾添加分号“；“
> 建议每行不超过 80 字符，如果超过，建议使用小括号将多行内容隐式地连接起来。
### 2.1.3.2 命名规范
> 1. 模块名尽量短小，全部小写，可以使用下划线来提高可读性；如：game_main
> 2. 包名尽量短小，全部小写，不建议使用下划线,如：com.example.game
> 3. 类名采用单词首字母的大写形式,如：BorrowBook
> 4. 模块内部的类采用下划线加类名，如：_BorrowBook
> 5. 函数、类的属性和方法的命名规则同模块类似，也是全部使用小写字母，多个字母之间用下划线连接，如：get_book_name
> 6. 常量命名时采用全部大写，可以使用下划线
> 7. 使用双下划线开头的实例变量或方法是类私有的。


# 2.2 保留字与标识符
## 2.2.1 保留字
> Python中的保留字是指在Python语言中已经被赋予特殊含义的单词，不能作为变量名、函数名或其他标识符的名称。

|and|as|assert|break|class|continue|
|----|----|----|----|----|----|
|def|del|elif|else|except|execfinally|
|for|from|False|global|if|import|
|in|is|lambda|nonlocal|not|none|
|or|pass|raise|return|try|True|
|while|with|yield|

> 注意：Python中的保留字是区分大小写的。
> Python中的保留字可以通过在python shell中输入代码来查看。
> ```python
> import keyword
> keyword.kwlist
> ```
## 2.2.2 标识符
> 标识符是指在Python语言中用于标识变量、函数、类、模块等的名称。
> 标识符的命名规则如下：
> 1. 标识符只能包含字母、数字和下划线“_”，不能以数字开头。
> 2. 标识符不能使用Python中的保留字作为名称。
> 3. 标识符的命名应该具有可读性，尽量使用有意义的名称。
> 4. 标识符的命名应该遵循大小写敏感的原则，即区分大小写。
> 5. python中以双下划线开头的标识符有特殊意义，一般应避免使用相似的标识符。
>

# 2.4 基本数据类型
## 2.4.1 数字类型
### 2.4.1.1 整数
>正整数、负整数、0
> 十进制、八进制、十六进制和二进制
>
> 八进制以0o或0O开头，如：0o10
> 十六进制以0x或0X开头，如：0x10

### 2.4.1.2 浮点数
> 浮点数是小数，可以用科学计数法表示，如：3.14e-2

### 2.4.1.3 复数
> 复数是由实数部分和虚数部分组成的数，可以用a+bi表示，其中a和b是实数部分，i是虚数单位。
>

## 2.4.2 字符串类型
> 字符串是用单引号或双引号括起来的字符序列。
> 字符串可以包含字母、数字、空格、标点符号等。
> 字符串可以使用三引号来表示多行字符串，如：'''多行字符串'''。
> 转义字符：使用反斜杠对一些特殊字符进行lating，如：\n表示换行符。
> 
> ***在字符串定界符引号的前面加上字母r，可以表示原始字符串，原始字符串中的转义字符不会被解释，如：r'原始字符串'。***


## 2.4.3 布尔类型
> 布尔类型只有两个值：True和False。
> 布尔类型可以用于条件判断，如：if语句。
> 布尔类型可以用于逻辑运算，如：and、or、not等。


## 2.4.4 数据类型转换
> 虽然在python中不需要显式地声明变量类型，但可以使用内置函数type()来查看变量的类型。
> 
> ***int(X)***：将X转换为整数类型
> ***float(X)***：将X转换为浮点数类型
> ***complex(real[,imag])***：创建一个复数，real是实数部分，imag是虚数部分，默认imag为0。
> ***str(X)***：将X转换为字符串类型
